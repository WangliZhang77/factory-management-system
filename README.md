# Factory Management System

A role-based factory work order management system built with **ASP.NET Core Razor Pages**, **Entity Framework Core**, and **PostgreSQL**.

This project is designed as a realistic junior-level backend/admin system, focusing on authentication, authorization, database design, and business rules commonly used in enterprise applications.

---

## Features

- ASP.NET Core Identity authentication
- Role-based access control (Admin / Supervisor / Operator)
- PostgreSQL database with EF Core migrations
- Secure login and user management
- Clean layered architecture (UI / Services / Domain / Infrastructure)

---

## Tech Stack

- .NET 8
- ASP.NET Core Razor Pages
- Entity Framework Core
- PostgreSQL
- ASP.NET Identity
- Git & GitHub
